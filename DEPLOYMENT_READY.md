## 🐛 Troubleshooting

### Build Fails
- Check `.npmrc` exists
- Run `npm run build` locally first
- Check Netlify build logs

### 404 Errors
- Verify `netlify.toml` exists
- Verify `public/_redirects` exists

### Features Not Working
- Check browser console (F12)
- Verify localStorage is enabled
- Try incognito mode
